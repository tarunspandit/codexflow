# CodexFlow website

The public product site for CodexFlow, built with Next-compatible React on
Cloudflare Sites.

## Prerequisites

- Node.js `>=22.13.0`

## Quick Start

```bash
npm install
npm run dev
npm run build
```

This starter does not use `wrangler.jsonc`.

## Useful Commands

- `npm run dev`: start local development
- `npm run build`: create the production Worker bundle
- `npm test`: build and verify English, Chinese, and social metadata output
- `npm run lint`: run the static quality gate
