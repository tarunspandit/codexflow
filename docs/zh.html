07常见问题

答案清楚。
没有玄学。

CodexFlow 会运行 Codex CLI 吗？+

不会。ChatGPT 提供模型和对话，CodexFlow 提供理解项目的本地工具后端。它不会启动、恢复或伪装 Codex session。

需要手动注册每个项目吗？+

不需要。直接运行 codexflow，它会自动发现最近的本地项目和配置的 roots。每个对话独立选择文件夹。

这是模型或额度代理吗？+

不是。它使用你自己的 ChatGPT 账号和已有产品限制。CodexFlow 不共享账号、不转售访问、不解锁模型，也不绕过额度。

多个对话可以同时工作吗？+

可以。一个 broker 和一个 tunnel 可以服务多个对话，每个 session 保留独立项目绑定。

安装需要什么？+

Node.js 20 或更高版本，以及支持 Apps / Developer Mode 的 ChatGPT 账号。常规启动流程已经内置 Cloudflare quick tunnel 支持。
