07QUESTIONS

Clear answers.
No mystique.

Does CodexFlow run the Codex CLI?+

No. ChatGPT supplies the model and conversation. CodexFlow supplies the project-aware local tool backend. It does not start, resume, or impersonate Codex sessions.

Do I register every project manually?+

No. The bare codexflow command discovers recent local project folders and configured roots automatically. Each conversation picks its own folder.

Is this a model or quota proxy?+

No. It uses your own ChatGPT account and its existing product limits. CodexFlow never pools accounts, resells access, unlocks models, or bypasses quotas.

Can several chats work at once?+

Yes. One broker and one tunnel can serve many conversations. Each session keeps an independent project binding.

What does installation require?+

Node.js 20 or newer and a ChatGPT account with Apps / Developer Mode access. Cloudflare quick tunnel support is built into the normal launcher flow.
